    Empfohlene (Online-) Ressourcen für studentische Side-Projects

    By Fabio Maienschein
    18/09/2020

     

    Studenten sind als zukünftige Arbeitskräfte und Entscheidungsträger ideale Zielgruppe für Anbieter von Software-Tools, sodass viele Anbieter dazu übergegangen sind, großzügige kostenlose Test- oder Free-Pläne zur Verfügung zu stellen, um diese von der Qualität dieser Angebote zu überzeugen. Solche Ressourcen sind ideal, um kleinere Projekte darauf aufzubauen und Erfahrung mit diesen Technologien zu erwerben. In dieser Lektion lernst du verschiedene Angebote und Ressourcen kennen, die du für dein nächstes Side-Project einsetzen kannst!

    1) Kostenlose Tools für Design und Entwicklung

    Folgende kostenlose Tools sind bei Designern und Developern besonders beliebt:

    Visual Studio Code: Der Open-Source Code Editor von Microsoft, der aufgrund seiner simplen, aber mächtigen Bedienoberfläche eine Allzweckwaffe für Entwicklungsprojekte ist und sich mittels Plugins nahezu beliebig anpassen lässt.

    Figma: Ein modernes Design-Tool für Anwendungen, über das Mockups für Benutzeroberflächen designed und über einen teilbaren Link als klickbarer Prototyp bereitgestellt werden kann. Figma setzt außerdem auf Kollaborationsfeatures, die einen gemeinsamen Design- und Reviewprozess wesentlich erleichtern.

    Git: Git hat sich in den letzten Jahren das populärste Tool zur Versionierung und Quellcodeverwaltung durchgesetzt und ist für alle Coding-Projekte grundlegend. Gehostete Quellcodeverwaltung wie GitHub und GitLab ermöglicht es, Source Code öffentlich (oder privat) abzulegen, und direkt auch zu hosten (GitHub Pages) oder als Grundlage für Continuous Integration/ Continuous Deployment (GitHub Actions).
    Die Hochschule München bietet außerdem eine eigengehostete GitLab-Instanz, die allen Angehörigen der Hochschule mihilfe ihrer Hochschulkennung offensteht.

    2) Kostenlose Angebote für Hosting und Cloud-Infrastruktur

    Folgende Anbieter empfehlen sich aufgrund ihrer großzügigen Free-Pläne für das Hosting kleinerer Projekte und Webseiten besonders gut:

    Netlify - extrem einfache Deployments von Frontend-Ressourcen (Webseiten und -applikationen) direkt aus der Quellcodeverwaltung heraus oder alternativ per Drag&Drop von lokalen Dateien.

    Firebase - Backend-as-a-Service Dienst von Google, der kostenlose Kontingente für Services wie Authentifizierung, Hosting und Storage (Firestore) bereitstellt.

    Heroku - Hosting von Web Applikationen (Free Plan mit Einschränkungen, Option eines kostenlosen Hobby-Dynos über GitHub Student)

    Amazon Webservices, Microsoft Azure, Google Cloud Platform - auch die großen Business-Clouds bieten Studenten Free-Plans, für die oftmals nur eine Registrierung mit Hochschuladresse (ohne Hinterlegen einer Kreditkarte) nötig ist.

    3) GitHub Student Developer Pack

    Als wohl größte Kollektion kostenloser Ressourcen für Software verdient das GitHub Student Developer Pack in jeder solchen Liste Erwähnung. Studenten mit gültiger Hochschuladresse erhalten nach ihrer Verifizierung durch GitHub eine Vielzahl von Angeboten. Unter anderem enthalten sind kostenlose Domänen (.me und .tech, beschränkt auf 1 Jahr), Cloud Ressourcen (AWS, Azure, Heroku), eine Kooperation mit Unity (kostenlose Assets), professionelle Entwicklungsumgebungen für eine Vielzahl von Programmiersprachen (Jetbrains) sowie ein- bis sechsmonatigen Zugänge für Tech-Kursplattformen (FrontendMasters, One Month, Flatiron School).

    Eine vollständige Übersicht über alle enthaltenen Freebies findet sich hier, außerdem lohnt sich ein Blick in die Gallerie von mithilfe des Packs umgesetzter Projekte.

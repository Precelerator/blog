 No-Code Tool Bubble: Web Apps erstellen

By Fabio Maienschein
18/09/2020

Egal ob TripAdvisor, AirBnB oder Shopify: Web-Apps und Plattformen stellen neben mobilen Apps die wichtigste Plattform für Consumer- und B2B Anwendungen dar. Bis vor wenigen Jahren war man in diesem Bereich noch auf aufwendige Webentwicklung angewiesen, immer mehr entstehen aber sogenannten No- bzw. Low-Code Plattformen, die auch technisch weniger versierten Nutzer die Erstellung von Webanwendungen ermöglichen.

Eine solche Low-Code Plattform ist Bubble, ein junges Startup aus New York. Die Plattform verspricht die vereinfachte Erstellung voll funktionsfähiger Webanwendungen mithilfe der hauseigenen visuellen Programmiersprache. Mithilfe von Bubble erstellte Webanwendungen werden von Bubble selbst gehostet und sind für Benutzer per URL zugreifbar.

Features

Bubble bietet verschiedene Features wie z.B. Authentifizierung (eine vorgeschaltete Anmeldung/Registrierung), Speichern, Abrufen und Bearbeiten von Daten, Anbindung von APIs, also Schnittstellen zu beliebigen Backends sowie die Anbindung von Zahlungsanbietren (z.B. Stripe). Erstellte Webapplikationen können responsive, also für alle Bildschirmgrößen geeignet gebaut werden, sodass sie auch auf Smartphones und Tablets passend dargestellt werden.

Prinzipien

Die Benutzeroberfläche von Bubble-Anwendungen kann aus einer Bibliothek aus Oberflächenkomponenten (Buttons, Textfelder, etc.) per Drag&Drop zusammengesetzt werden. Sogenannte Workflows regeln die Abläufe, die bei Benutzerinteraktionen von der Anwendung ausgeführt werden sollen.
